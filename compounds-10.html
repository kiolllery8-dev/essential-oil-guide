CHEMICAL FAMILY X

酚與芳香醛類

酚類（Phenols）是精油中最強效的抗菌成分，以百里酚（Thymol）、香芹酚（Carvacrol）、丁香酚（Eugenol）為代表。芳香醛類（Aromatic Aldehydes）如肉桂醛（Cinnamaldehyde）同樣具有強力抗感染作用。此類精油效力強大但具皮膚刺激性，使用濃度須嚴格控制。

246
中國肉桂
Cinnamomum cassia
主要化學成分
肉桂醛 Cinnamaldehyde (80-90%)、香豆素 Coumarin (3%)、乙酸肉桂酯
萃取部位 / 科屬
樹皮及帶葉枝幹 / 樟科 Lauraceae
主要功效
強力抗菌、抗病毒、促進血液循環、暖身驅寒、補強神經、緩解肌肉緊張、助消化
皮膚刺激孕婦禁用抗菌暖身
高度皮膚刺激性，接觸皮膚最高安全濃度 0.05%。懷孕期間避用，兒童不可使用。含高量香豆素，肝臟疾病者慎用。
247
台灣土肉桂
Cinnamomum osmophloeum
主要化學成分
肉桂醛 Cinnamaldehyde (40-81%)、芳樟醇 Linalool、丁香酚 Eugenol、1,8-桉油醇
萃取部位 / 科屬
葉片 / 樟科 Lauraceae
主要功效
抗菌、抗白蟻、降血糖、降血脂、抗發炎、抗氧化、美白
皮膚刺激抗菌台灣特有種
含肉桂醛具皮膚刺激性，使用濃度需低於 0.5%。孕婦避用。因化學型態多變，使用前需確認成分比例。
248
印度肉桂
Cinnamomum tamala
主要化學成分
丁香酚 Eugenol、芳樟醇 Linalool、石竹烯 Caryophyllene、乙酸丁香酯、肉桂醛（依化學型而異）
萃取部位 / 科屬
葉片 / 樟科 Lauraceae
主要功效
降血糖、降血脂、消炎、養肝、健胃、助循環、局部暖身、舒緩肌肉骨骼疼痛
消炎助消化暖身
相較其他肉桂精油刺激性較低，但仍建議低濃度使用（1% 以下）。孕婦及幼兒慎用。
249
錫蘭肉桂
Cinnamomum verum
主要化學成分
樹皮：肉桂醛 Cinnamaldehyde (65-80%)、乙酸肉桂酯；葉片：丁香酚 Eugenol (70-96%)、石竹烯
萃取部位 / 科屬
樹皮或葉片 / 樟科 Lauraceae
主要功效
強效抗菌、抗病毒、抗真菌、促進循環、暖身、助消化、提振免疫、抗氧化
皮膚刺激孕婦禁用抗菌暖身
皮膚刺激性強，樹皮精油安全使用濃度 0.07%。葉片精油（丁香酚型）較溫和但仍需稀釋。孕婦及兒童避用。
250
頭狀百里香
Thymbra capitata
主要化學成分
香芹酚 Carvacrol (60-73%)、對繖花烴 p-Cymene、γ-萜品烯 γ-Terpinene、百里酚 Thymol
萃取部位 / 科屬
開花頂端 / 唇形科 Lamiaceae
主要功效
強效抗菌、抗真菌、抗病毒、抗寄生蟲、免疫刺激、消炎
皮膚刺激抗菌抗真菌
高含量酚類成分，皮膚刺激性強。使用濃度不超過 1%。孕婦、幼兒、高血壓患者避用。
251
小茴香
Cuminum cyminum
主要化學成分
小茴香醛 Cuminaldehyde (27-45%)、β-蒎烯 β-Pinene (25%)、γ-萜品烯 γ-Terpinene (15%)、對繖花烴
萃取部位 / 科屬
種子 / 繖形科 Apiaceae
主要功效
助消化、驅風、抗菌、抗氧化、鎮靜神經、促進食慾
光敏性助消化驅風
具光敏性，使用後 12 小時內避免日曬。皮膚敏感者低濃度使用。孕婦慎用。
252
丁香花苞
Syzygium aromaticum
主要化學成分
丁香酚 Eugenol (72-89%)、乙酸丁香酯 Eugenyl Acetate (5-15%)、β-石竹烯 β-Caryophyllene (5-14%)
萃取部位 / 科屬
花苞 / 桃金孃科 Myrtaceae
主要功效
強效抗菌、抗病毒、止痛（尤其牙痛）、抗寄生蟲、促進循環、助消化、抗氧化
皮膚刺激止痛抗菌牙痛
丁香酚具黏膜刺激性，皮膚使用濃度建議 0.5% 以下。避免長期大量使用。孕婦慎用，可能影響凝血功能。
253
丁香羅勒
Ocimum gratissimum
主要化學成分
丁香酚 Eugenol (43-69%)、甲基丁香酚 Methyl Eugenol、順式-羅勒烯 cis-Ocimene、γ-木羅烯
萃取部位 / 科屬
葉片及花頂 / 唇形科 Lamiaceae
主要功效
抗菌、抗真菌、抗病毒、抗發炎、驅蟲、助消化、退燒
皮膚刺激抗菌驅蟲
含高比例丁香酚，具皮膚刺激性。使用濃度不超過 1%。含甲基丁香酚，長期大量使用有安全疑慮。孕婦避用。
254
神聖羅勒
Ocimum tenuiflorum
主要化學成分
丁香酚 Eugenol (30-60%)、β-石竹烯 β-Caryophyllene (4-45%)、β-欖香烯 β-Elemene、甲基丁香酚
萃取部位 / 科屬
葉片及花頂 / 唇形科 Lamiaceae
主要功效
適應原（抗壓力）、抗菌、抗氧化、抗發炎、免疫調節、降血糖、呼吸道保健
適應原抗壓力免疫神聖植物
丁香酚具皮膚刺激性，建議稀釋使用。孕婦慎用。可能與抗凝血藥物產生交互作用。
255
野馬郁蘭
Origanum vulgare
主要化學成分
香芹酚 Carvacrol (30-70%)、百里酚 Thymol (2-19%)、對繖花烴 p-Cymene (5-11%)、γ-萜品烯
萃取部位 / 科屬
開花頂端 / 唇形科 Lamiaceae
主要功效
最強效天然抗生素之一、抗菌、抗病毒、抗真菌、抗寄生蟲、免疫刺激、消炎止痛
皮膚刺激孕婦禁用天然抗生素抗感染
高酚類含量，皮膚刺激性極強。使用濃度不超過 1%，短期使用為宜。孕婦、幼兒、高血壓及肝病患者禁用。

萃取部位 / 科屬

主要功效
抗菌、抗真菌、抗氧化、抗痙攣、助消化、驅風、抗腹瀉、抗發炎
皮膚刺激助消化抗菌
含酚類成分，具皮膚刺激性。建議稀釋至 1% 以下使用。孕婦慎用。

希臘野馬郁蘭
Origanum heracleoticum
主要化學成分
香芹酚 Carvacrol (70-86%)、百里酚 Thymol (3-8%)、對繖花烴 p-Cymene、γ-萜品烯
萃取部位 / 科屬
開花頂端 / 唇形科 Lamiaceae
主要功效
頂級抗感染精油、抗菌、抗病毒、抗真菌、免疫刺激、抗寄生蟲
皮膚刺激孕婦禁用抗感染高香芹酚
香芹酚含量極高，皮膚刺激性極強。使用濃度不超過 1%，必須大量稀釋。孕婦、哺乳期、幼兒、高血壓患者禁用。

萃取部位 / 科屬
果實或葉片 / 桃金孃科 Myrtaceae
主要功效
止痛、抗菌、助消化、驅風、暖身、促進循環、抗痙攣
皮膚刺激止痛助消化
高丁香酚含量具黏膜刺激性。使用濃度不超過 0.5%。孕婦慎用。可能影響凝血功能。

西印度月桂
Pimenta racemosa
主要化學成分
丁香酚 Eugenol (50-60%)、月桂烯 Myrcene (20%)、蒔蘿腦 Chavicol (6%)、桉油醇、α-萜品醇
萃取部位 / 科屬
葉片 / 桃金孃科 Myrtaceae
主要功效
止痛、促進頭髮生長、抗菌、促進循環、暖身、提振精神、緩解肌肉疼痛
皮膚刺激護髮止痛暖身
含丁香酚及蒔蘿腦，具皮膚敏感性。使用濃度建議 1% 以下。孕婦慎用。
260
到手香
Plectranthus amboinicus
主要化學成分
香芹酚 Carvacrol (28-89%)、百里酚 Thymol (21-62%)、γ-萜品烯、對繖花烴 p-Cymene（因化學型而異）
萃取部位 / 科屬
葉片 / 唇形科 Lamiaceae
主要功效
抗菌、抗真菌、驅蟲、抗氧化、抗發炎、助消化、呼吸道保健、防黴
皮膚刺激抗菌驅蟲防黴
含高比例酚類成分，具皮膚刺激性。化學型差異大，使用前需確認成分。稀釋至 1% 以下使用。孕婦慎用。
261
重味過江藤
Lippia origanoides
主要化學成分
香芹酚 Carvacrol (33-44%)、百里酚 Thymol (5-11%)、對繖花烴 p-Cymene (12-16%)、甲基百里酚 (6-9%)
萃取部位 / 科屬
葉片及開花頂端 / 馬鞭草科 Verbenaceae
主要功效
強效抗菌、抗真菌、抗氧化、防腐保鮮、口腔保健
皮膚刺激抗菌抗氧化
含酚類成分，皮膚刺激性中等。化學型多變（香芹酚型/百里酚型/樟腦型），使用前需確認。孕婦慎用。
262
黑種草
Nigella sativa
主要化學成分
百里醌 Thymoquinone (30-63%)、對繖花烴 p-Cymene (31%)、α-側柏烯 α-Thujene (6%)、百里醇氫醌
萃取部位 / 科屬
種子 / 毛茛科 Ranunculaceae
主要功效
強效抗氧化、免疫調節、抗發炎、抗菌、抗腫瘤輔助、支持呼吸道、護肝、降血糖
免疫調節抗氧化護肝傳統藥用
百里醌具生物活性，高濃度可能刺激皮膚。孕婦慎用。可能與降血糖藥物產生交互作用。

希臘香薄荷
Satureja thymbra
主要化學成分
香芹酚 Carvacrol (4-40%) 或百里酚 Thymol (25-66%)（依化學型）、γ-萜品烯 (23-39%)、對繖花烴 p-Cymene
萃取部位 / 科屬
開花全草 / 唇形科 Lamiaceae
主要功效
抗菌、抗真菌、抗氧化、抗增生、免疫刺激、消炎止痛
皮膚刺激抗菌抗氧化
含酚類成分，具皮膚刺激性。化學型因海拔差異明顯（低地多香芹酚型，高地多百里酚型）。使用濃度 1% 以下。孕婦慎用。
265
百里酚百里香
Thymus vulgaris ct. Thymol
主要化學成分
百里酚 Thymol (23-60%)、對繖花烴 p-Cymene (8-44%)、γ-萜品烯 γ-Terpinene (6-18%)、香芹酚 Carvacrol (2-8%)
萃取部位 / 科屬
開花全草 / 唇形科 Lamiaceae
主要功效
強效抗菌（呼吸道感染）、祛痰、抗真菌、免疫刺激、促進循環、驅蟲、抗疲勞提振
皮膚刺激孕婦禁用呼吸道免疫
百里酚是最烈性的百里香化學型。皮膚刺激性強，使用濃度不超過 1%。孕婦、幼兒、高血壓及甲亢患者禁用。
266
野地百里香
Thymus serpyllum
主要化學成分
香芹酚 Carvacrol (27-48%) 或百里酚 Thymol (5-19%)、對繖花烴 p-Cymene (9-19%)、甲基香芹酚、香葉醇（依產地而異）
萃取部位 / 科屬
開花全草 / 唇形科 Lamiaceae
主要功效
抗菌、祛痰、止咳、助消化、驅風、抗痙攣、提振精神、鎮靜安眠（因化學型而異）
皮膚刺激呼吸道助消化
化學組成因產地差異極大。含酚類化學型具皮膚刺激性，使用濃度 1% 以下。北歐產區不含酚類，較為溫和。孕婦慎用。

印度藏茴香
Trachyspermum ammi
主要化學成分
百里酚 Thymol (50-72%)、γ-萜品烯 γ-Terpinene (13-48%)、對繖花烴 p-Cymene (14-34%)、香芹酚
萃取部位 / 科屬
種子 / 繖形科 Apiaceae
主要功效
強效抗菌、抗真菌、驅風、助消化、抗痙攣、驅蟲、呼吸道保健、解毒
皮膚刺激孕婦禁用助消化驅蟲
