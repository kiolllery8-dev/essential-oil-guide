倍半萜醇類

倍半萜醇（Sesquiterpene Alcohols）是 15 碳骨架搭配羥基的醇類化合物，分子量較大、揮發速度慢，多為精油的基調。代表成分包括檀香醇（Santalol）、廣藿香醇（Patchoulol）、岩蘭草醇（Vetiverol）等。此類精油氣味沉穩厚實，具有優異的皮膚修復、抗發炎與鎮靜安神效果，整體安全性高。

阿米香樹
Amyris balsamifera
主要化學成分
纈草醇 Valerianol (30-50%)、欖香醇 Elemol、γ-桉葉醇 γ-Eudesmol、α-桉葉醇、β-桉葉醇
萃取部位 / 科屬
木材 / 芸香科 Rutaceae
主要功效
鎮靜安神、助眠、靜脈循環、抗痙攣、定香劑、皮膚保濕
助眠鎮靜定香平價檀香替代
整體安全性高，適合日常使用。氣味近似檀香，常作為平價替代品。孕婦可低劑量使用。
271
沉香樹
Aquilaria malaccensis
主要化學成分
沉香醇 Agarospirol、β-桉葉醇 β-Eudesmol、γ-桉葉醇、薁 Azulene 衍生物、色酮類 Chromones
萃取部位 / 科屬
含樹脂的心材 / 瑞香科 Thymelaeaceae
主要功效
深度鎮靜安神、冥想輔助、抗發炎、皮膚修復、助消化、平衡神經系統
冥想鎮靜珍稀靈性用油
安全性高，但極為珍稀昂貴。確保來源合法（CITES 管制物種）。偽品眾多，需謹慎選購。
272
玉檀木
Bulnesia sarmientoi
主要化學成分
癒創木醇 Guaiol (42-72%)、布蘭瑟醇 Bulnesol (25%)、桉葉醇 Eudesmol、欖香醇 Elemol
萃取部位 / 科屬
心材木屑 / 蒺藜科 Zygophyllaceae
主要功效
鎮靜安神、消炎、靜脈循環促進、皮膚修復、抗菌、定香劑
鎮靜消炎定香CITES管制
安全性高，適合日常使用。室溫下呈蠟狀固態，需加溫使用。已列入 CITES 附錄 II，需確認合法來源。
273
降香
Dalbergia odorifera
主要化學成分
反式-橙花叔醇 trans-Nerolidol (30%)、橙花叔醇氧化物 Nerolidol Oxides (合計 55-60%)、倍半萜醇類
萃取部位 / 科屬
心材 / 豆科 Fabaceae
主要功效
活血化瘀、止痛、心肌保護、抗氧化、鎮靜安神、抗發炎
活血化瘀中藥材心血管珍稀
中醫傳統藥材，安全性中等。孕婦及出血體質者慎用（活血作用）。珍稀物種，需確認來源。
274
胡蘿蔔籽
Daucus carota
主要化學成分
胡蘿蔔醇 Carotol (19-76%)、石竹烯 Caryophyllene (5-13%)、α-蒎烯 α-Pinene、乙酸香葉酯 Geranyl Acetate
萃取部位 / 科屬
種子 / 繖形科 Apiaceae
主要功效
卓越皮膚修復與再生、護肝解毒、腎臟保健、抗老化、調理膚色、促進細胞更新
皮膚修復護肝抗老化排毒
安全性高，適合日常護膚使用。芳療界公認的頂級護膚精油。孕婦前三個月慎用。
275
暹羅木
Fokienia hodginsii

柏木醇 Cedrol、焦杉醇 Fokienol、α-桉葉醇 α-Eudesmol、柏木烯 Cedrene
萃取部位 / 科屬
木材 / 柏科 Cupressaceae
主要功效
鎮靜安神、促進淋巴循環、消炎、呼吸道保健、皮膚調理
鎮靜淋巴循環木質調
安全性高，適合日常使用。氣味似雪松，木質溫暖。孕婦可低劑量使用。
276
白草果
Aframomum daniellii
主要化學成分
桉油醇 1,8-Cineole、β-蒎烯 β-Pinene、倍半萜醇類、石竹烯 β-Caryophyllene
萃取部位 / 科屬
種子 / 薑科 Zingiberaceae
主要功效
助消化、驅風、抗菌、抗氧化、暖身、止痛
助消化暖身香料精油
安全性中等，建議稀釋使用。孕婦慎用。
277
黏答答土木香
Inula graveolens
主要化學成分
龍腦乙酸酯 Bornyl Acetate、土木香內酯 Alantolactone、倍半萜醇類、桉油醇 1,8-Cineole
萃取部位 / 科屬
開花全草 / 菊科 Asteraceae
主要功效
強效化黏液、呼吸道保健（支氣管炎）、抗卡他性炎、祛痰、抗菌
致敏風險呼吸道化黏液
菊科植物可能引起過敏反應。內酯成分可能致敏。使用前需做皮膚測試。孕婦、幼兒慎用。
278
昆士亞
Kunzea ambigua
主要化學成分
α-蒎烯 α-Pinene、1,8-桉油醇 1,8-Cineole、紅沒藥醇 Bisabolol、石竹烯 Caryophyllene、欖香醇
萃取部位 / 科屬
葉片及嫩枝 / 桃金孃科 Myrtaceae
主要功效
消炎止痛、舒緩肌肉關節、抗菌、促進傷口癒合、情緒舒緩
消炎止痛澳洲原生
安全性高，澳洲原住民傳統用油。適合日常使用。孕婦可低劑量使用。
279
厚朴
Magnolia officinalis
主要化學成分
厚朴酚 Magnolol、和厚朴酚 Honokiol、β-桉葉醇 β-Eudesmol、石竹烯 Caryophyllene
萃取部位 / 科屬
樹皮 / 木蘭科 Magnoliaceae
主要功效
深度鎮靜安神、抗焦慮、抗發炎、助消化、抗菌、抗腫瘤輔助、肌肉放鬆
鎮靜抗焦慮中藥材安神
中醫傳統藥材。厚朴酚具有肌肉鬆弛作用，高劑量可能引起嗜睡。孕婦慎用。

橙花叔醇綠花白千層
Melaleuca quinquenervia ct. Nerolidol
主要化學成分
反式-橙花叔醇 trans-Nerolidol (80-90%)、沉香醇 Linalool、α-萜品醇 α-Terpineol
萃取部位 / 科屬
葉片 / 桃金孃科 Myrtaceae
主要功效
皮膚修復與再生、抗發炎、抗真菌、雌激素樣作用、靜脈保健、放射線損傷修護
皮膚修復抗真菌類雌激素
可能具有類雌激素作用，荷爾蒙敏感族群（乳癌、子宮內膜異位等）慎用。孕婦避用。

香脂果豆木
Myroxylon balsamum
主要化學成分
苯甲酸苄酯 Benzyl Benzoate、肉桂酸苄酯 Benzyl Cinnamate、橙花叔醇 Nerolidol、香草醛 Vanillin
萃取部位 / 科屬
樹脂 / 豆科 Fabaceae
主要功效
皮膚修復、抗菌、促進傷口癒合、祛痰、止咳、定香劑
致敏風險皮膚修復定香
苯甲酸苄酯可能引起接觸性皮膚炎。過敏體質者需做皮膚測試。IFRA 限制使用濃度。

新喀里多尼亞松
Neocallitropsis pancheri
主要化學成分
癒創木醇 Guaiol、布蘭瑟醇 Bulnesol、桉葉醇 Eudesmol、倍半萜烯類
萃取部位 / 科屬
木材 / 柏科 Cupressaceae
主要功效
消炎、鎮靜、皮膚保健、靜脈循環促進、抗菌
消炎鎮靜珍稀新喀里多尼亞特有
安全性高。極為珍稀的特有種精油，市場上稀少。
283
羌活
Notopterygium incisum
主要化學成分
羌活醇 Notopterol、異歐前胡素 Isoimperatorin、倍半萜醇類、β-桉葉醇
萃取部位 / 科屬
根莖 / 繖形科 Apiaceae
主要功效
祛風散寒、止痛（頭痛、關節痛）、抗發炎、解表、利濕
止痛祛風中藥材
中醫傳統藥材，含香豆素類成分，可能有光敏性。使用後避免日曬。孕婦慎用。陰虛血虧者不宜。
284
草澄茄
Litsea cubeba
主要化學成分
檸檬醛 Citral (60-80%)、沉香醇 Linalool、橙花叔醇 Nerolidol、α-萜品醇
萃取部位 / 科屬
果實 / 樟科 Lauraceae
主要功效
抗菌、抗真菌、抗發炎、助消化、驅蟲、提振情緒、降血壓
皮膚敏感抗菌提振檸檬香
檸檬醛含量高，可能引起皮膚敏感。使用濃度建議 1% 以下。敏感肌膚需做皮膚測試。孕婦慎用。
285
廣藿香
Pogostemon cablin
主要化學成分
廣藿香醇 Patchoulol (26-40%)、α-癒創木烯 α-Guaiene、α-布蘭瑟烯、β-石竹烯、廣藿香酮 Pogostone
萃取部位 / 科屬
葉片 / 唇形科 Lamiaceae
主要功效
卓越皮膚修復（疤痕、妊娠紋）、抗發炎、促進細胞再生、靜脈循環、抗菌、助消化、情緒安定
皮膚修復抗老化定香接地
安全性極高，適合各年齡層使用。陳年精油品質更佳。定香效果優異。孕婦可低劑量使用。
286
狹長葉鼠尾草
Salvia stenophylla
主要化學成分
沉香醇 Linalool、α-紅沒藥醇 α-Bisabolol、倍半萜醇類、1,8-桉油醇
萃取部位 / 科屬
全草 / 唇形科 Lamiaceae
主要功效
皮膚保健、抗菌、抗氧化、消炎、傷口癒合
皮膚保健抗菌南非原生
安全性高。南非特有鼠尾草品種，不含側柏酮，較一般鼠尾草安全。
287
檀香
Santalum album
主要化學成分
α-檀香醇 α-Santalol (41-55%)、β-檀香醇 β-Santalol (16-24%)、倍半萜醇總含量 >90%
萃取部位 / 科屬
心材 / 檀香科 Santalaceae
主要功效
皮膚修復（乾燥、老化、痘痘）、深度鎮靜安神、泌尿道保健、抗發炎、冥想輔助、催情
皮膚修復鎮靜冥想珍稀催情
安全性極高，幾乎無禁忌。珍稀昂貴（印度產區嚴格管制），偽品極多需謹慎選購。孕婦可使用。
288
太平洋檀香
Santalum austrocaledonicum
主要化學成分
α-檀香醇 α-Santalol、β-檀香醇 β-Santalol、Z-蘭瑟醇 Z-Lanceol、倍半萜醇類
萃取部位 / 科屬
心材 / 檀香科 Santalaceae
主要功效
皮膚修復、鎮靜安神、抗發炎、泌尿道保健、催情、冥想輔助
皮膚修復鎮靜太平洋產區
安全性極高，與印度檀香功效相近。新喀里多尼亞及萬那杜產區。品質優異的印度檀香替代品。
289
塔斯馬尼亞胡椒
Tasmannia lanceolata
主要化學成分
多瀕二醛 Polygodial、α-蒎烯 α-Pinene、β-蒎烯、沙賓烯 Sabinene、石竹烯 Caryophyllene、大根香葉烯 D
萃取部位 / 科屬
葉片及果實 / 林仙科 Winteraceae
主要功效
強效抗菌、抗真菌、消炎止痛、防腐保鮮、驅蟲、抗氧化
皮膚刺激抗菌澳洲原生辛辣
多瀕二醛具有辛辣刺激性，可能刺激皮膚及黏膜。使用需大量稀釋。孕婦慎用。
290
纈草
Valeriana officinalis
主要化學成分
乙酸龍腦酯 Bornyl Acetate (2-36%)、纈草醛 Valerenal、纈草烯酸 Valerenic Acid、纈草酮 Valeranone
萃取部位 / 科屬
根部 / 敗醬科 Valerianaceae
主要功效
強效鎮靜安眠、抗焦慮、抗痙攣、降血壓、神經系統修護、緩解緊張性頭痛
助眠鎮靜抗焦慮神經修護
鎮靜效果強，可能引起嗜睡，不宜在需要專注時使用。氣味特殊（土味/貓尿味）。不建議與鎮靜劑同用。孕婦慎用。
291
印度纈草
Valeriana jatamansi
主要化學成分
廣藿香醇 Patchoulol、馬兜鈴烯 Aristolene、纈草醛 Valerenal、異纈草酸 Isovaleric Acid、倍半萜醇類
萃取部位 / 科屬
根莖 / 敗醬科 Valerianaceae
主要功效
深度鎮靜安神、抗焦慮、抗癲癇輔助、皮膚修復、接地安定、冥想輔助
鎮靜安神接地喜馬拉雅
鎮靜效果強，注意嗜睡風險。又稱甘松或穗甘松，喜馬拉雅山區植物。不建議與鎮靜劑同用。孕婦慎用。
292
岩蘭草
Vetiveria zizanioides
主要化學成分
岩蘭草醇 Vetiverol (18-50%)、岩蘭草酮 Vetivone、纈草醛 Valerenal、β-岩蘭草烯 β-Vetivenene、倍半萜醇類 >70%
萃取部位 / 科屬
根部 / 禾本科 Poaceae
主要功效
深度接地安定、鎮靜安神、促進循環、皮膚修復（痘疤、妊娠紋）、免疫調節、抗氧化
接地鎮靜皮膚修復定香
安全性極高，幾乎無禁忌。陳年後品質更佳，氣味更圓潤。極佳的基調定香劑。孕婦可使用。
