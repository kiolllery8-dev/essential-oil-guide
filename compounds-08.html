倍半萜酮類

倍半萜酮分子量較大、親膚性佳，護膚作用優於單萜酮且氣味甜美溫和。具有促進細胞再生、化瘀消腫、黏液溶解等特性，高劑量使用仍相對安全。代表性精油包括雪松、永久花、薑黃與松紅梅等。

德州白蘿蔔
Ammi visnaga
主要化學成分
異戊基-2-甲基丁酸酯、異戊基異丁酸酯、沉香醇、凱林（Khellin）、維斯納金（Visnagin）
萃取部位 / 科屬
種子（果實） / 繖形科
主要功效
支氣管擴張、緩解氣喘、抗痙攣、擴張冠狀動脈、利尿、緩解腎結石
呼吸道抗痙攣泌尿系統
含呋喃香豆素成分，使用後避免日曬；孕婦及低血壓者慎用。
199
鑽石
Callitris intratropica
主要化學成分
癒創木醇（Guaiol）、癒創木烯（Guaiazulene）、β-桉葉醇、倍半萜酮類
萃取部位 / 科屬
木材 / 柏科
主要功效
鎮靜安神、抗菌防腐、促進傷口癒合、驅蟲、淨化空氣
鎮靜抗菌木質調
一般劑量下安全使用；孕婦建議低劑量。
200
印蒿
Artemisia pallens
主要化學成分
順式-印蒿酮（cis-Davanone）43%、橙花醇 10%、反式-印蒿酮 5%、異印蒿酮
萃取部位 / 科屬
花朵與全草 / 菊科
主要功效
調理經期、緩解更年期不適、抗病毒、抗真菌、促進皮膚再生、安撫助眠
婦科調理皮膚再生助眠
氣味濃烈，建議低劑量使用；孕婦避免使用。
201
大西洋雪松
Cedrus atlantica
主要化學成分
大西洋酮（Atlantone）、β-喜馬雪松烯、α-喜馬雪松烯、γ-喜馬雪松烯
萃取部位 / 科屬
木材 / 松科
主要功效
化痰排黏液、促進淋巴循環、消脂緊膚、助眠安神、驅蟲防蛀
淋巴循環化痰緊膚助眠
孕婦與嬰幼兒避免使用（含酮類）。
202
喜馬拉雅雪松
Cedrus deodara
主要化學成分
大西洋酮（Atlantone）、α-喜馬雪松烯、β-喜馬雪松烯、雪松醇
萃取部位 / 科屬
木材 / 松科
主要功效
促進淋巴代謝、化痰止咳、緩解關節炎、養髮護髮、安定神經
淋巴代謝關節保養護髮
孕婦與嬰幼兒避免使用（含酮類）。
203
杭白菊
Chrysanthemum morifolium
主要化學成分
樟腦（Camphor）、1,8-桉油醇、龍腦、乙酸龍腦酯、β-薑黃烯、α-沒藥醇
萃取部位 / 科屬
花朵 / 菊科
主要功效
清熱明目、抗氧化、抗菌消炎、舒緩頭痛、鎮靜安神
清熱明目抗氧化
一般劑量下安全；菊科過敏者慎用。
204
薑黃
Curcuma longa
主要化學成分
薑黃酮（Turmerone）60%、芳薑黃酮（ar-Turmerone）、薑黃烯（Curcumene）、薑烯
萃取部位 / 科屬
根莖 / 薑科
主要功效
強效抗氧化、抗發炎、促進消化、護肝、促進傷口癒合、抗腫瘤
抗發炎抗氧化護肝消化
孕婦避免大量使用；可能刺激敏感肌膚，建議稀釋至 1% 以下。
205
莪朮
Curcuma zedoaria
主要化學成分
莪朮醇（Curcumol）、蓬莪朮酮、吉馬酮（Germacrone）、莪朮烯、β-欖香烯
萃取部位 / 科屬
根莖 / 薑科
主要功效
行氣活血、化瘀止痛、消積散結、抗腫瘤、護肝
活血化瘀止痛消積
孕婦禁用（強效活血化瘀）；月經過多者慎用。
206
莎草
Cyperus rotundus
主要化學成分
α-香附酮（α-Cyperone）、莎草酮（Cyperotundone）、齊墩酮（Zierone）、β-蛇床烯
萃取部位 / 科屬
根莖（塊莖） / 莎草科
主要功效
疏肝理氣、調經止痛、行氣消脹、抗菌抗氧化
理氣調經抗菌
孕婦慎用；氣虛無滯者不宜使用。
207
義大利永久花
Helichrysum italicum
主要化學成分
義大利雙酮（Italidione）、乙酸橙花酯、橙花醇、γ-薑黃烯、α-蒎烯
萃取部位 / 科屬
花朵 / 菊科
主要功效
強效化瘀消腫、促進細胞再生、修復疤痕、抗發炎、護肝、抗過敏
化瘀疤痕修復抗發炎護肝
安全性高，一般劑量下無禁忌；菊科過敏者首次少量測試。
208
大根老鸛草
Geranium macrorrhizum
主要化學成分
大根老鸛草酮（Germacrone）、α-沒藥醇、α-桉葉醇、γ-欖香烯、δ-癒創木烯
萃取部位 / 科屬
全草 / 牻牛兒苗科
主要功效
消炎抗菌、平衡雌激素、促進淋巴循環、舒緩濕疹、改善油性膚質
荷爾蒙平衡抗菌皮膚調理
一般劑量下安全；荷爾蒙敏感者建議諮詢專業人士。
209
薰衣草
Lavandula stoechas
主要化學成分
葑酮（Fenchone）、樟腦（Camphor）、1,8-桉油醇、龍腦、α-蒎烯
萃取部位 / 科屬
花朵與全草 / 唇形科
主要功效
化痰排黏液、促進傷口癒合、殺菌消炎、驅蟲、頭痛緩解
化痰傷口癒合含酮量高
此為頭狀薰衣草，酮類含量高，禁用於孕婦、嬰幼兒及癲癇患者；不可與真正薰衣草混淆。
210
馬纓丹
Lantana camara
主要化學成分
β-丁香油烴（β-Caryophyllene）23%、α-蛇麻烯 11%、大根香葉烯 D 10%、印蒿酮
萃取部位 / 科屬
葉片 / 馬鞭草科
主要功效
抗菌消炎、驅蟲防蚊、鎮靜舒緩、退熱、舒緩呼吸道不適
抗菌驅蟲鎮靜
全株有毒，精油須由專業人員調配使用；孕婦、兒童及寵物禁用。
211
松紅梅
Leptospermum scoparium
主要化學成分
細籽酮（Leptospermone）12–18%、黃酮（Flavesone）、異細籽酮、(Z)-石竹烯
萃取部位 / 科屬
葉片與嫩枝 / 桃金孃科
主要功效
強效抗菌抗真菌、促進傷口癒合、抗過敏、舒緩皮膚炎症、私密護理
抗菌傷口癒合皮膚護理
安全性高，溫和不刺激；一般劑量下無特殊禁忌。
212
桂花
Osmanthus fragrans
主要化學成分
反式-β-紫羅蘭酮、沉香醇、牻牛兒醇、γ-癸內酯、二氫-β-紫羅蘭酮
萃取部位 / 科屬
花朵（溶劑萃取） / 木樨科
主要功效
安眠舒壓、紓解經前症候群、抗氧化、撫平皺紋、提神醒腦
安眠舒壓抗老化
安全性高，氣味溫和；溶劑萃取品建議外用為主。
213
紫羅蘭
Viola odorata
主要化學成分
α-紫羅蘭酮（α-Ionone）、巴馬酮（Parmone）、壬二烯醇、苯甲醇、己醇
萃取部位 / 科屬
葉片（溶劑萃取） / 堇菜科
主要功效
舒緩緊張情緒、抗發炎、改善失眠、護膚保濕、緩解頭痛
舒壓助眠護膚
安全性高；溶劑萃取品適合低劑量外用與擴香。
