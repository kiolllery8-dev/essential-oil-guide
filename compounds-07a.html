脂類 Esters

脂類（酯類）是精油中最常見的芳香分子家族之一，由醇類與酸類縮合而成。脂類精油通常具有令人愉悅的果香或花香氣味，以溫和的抗痙攣、鎮靜神經、抗發炎與平衡情緒著稱，是芳療中安全性最高的化學族群之一。

共收錄 17 種精油（#156 ~ #172）

156
同密助
Dipteryx odorata
主要化學成分
香豆素 (60~90%)、肉桂酸衍生物、香草醛、苯甲醛
萃取部位 / 科屬
種子（豆莢） / 豆科 Fabaceae
主要功效
鎮靜安撫、抗痙攣、淋巴引流、舒緩呼吸道、增添溫暖香氣基調
鎮靜香氣定香溫暖
含高量香豆素，具光敏性，使用後避免陽光曝曬；口服有肝毒性風險，僅限外用且低劑量。孕婦及兒童避免使用。
157
羅馬洋甘菊
Chamaemelum nobile
主要化學成分
當歸酸異丁酯 (13~36%)、當歸酸異戊酯 (8~30%)、3-甲基戊基當歸酸酯、松香芹酮
萃取部位 / 科屬
花朵（蒸餾） / 菊科 Asteraceae
主要功效
強效抗痙攣、鎮靜神經、助眠、抗發炎、緩解消化不適、安撫情緒躁動
抗痙攣助眠嬰幼兒適用消化
安全性極高，適合嬰幼兒及孕婦低劑量使用。菊科過敏者需先做皮膚測試。
158
墨西哥沉香
Bursera delpechiana
主要化學成分
沉香醇 (30~50%)、乙酸沉香酯 (20~40%)、乙酸香葉酯、香葉醇
萃取部位 / 科屬
木材 / 橄欖科 Burseraceae
主要功效
抗焦慮、鎮靜放鬆、抗菌、皮膚修護、調理油性肌膚
抗焦慮皮膚修護木質調
一般安全，低劑量適用於多數族群。
159
苦橙葉
Citrus aurantium (leaves)
主要化學成分
乙酸沉香酯 (47~58%)、沉香醇 (20~25%)、乙酸香葉酯 (3~5%)、α-松油醇
萃取部位 / 科屬
葉片（蒸餾） / 芸香科 Rutaceae
主要功效
抗痙攣、鎮靜安眠、平衡自律神經、緩解焦慮與恐慌、皮膚再生
抗痙攣安眠自律神經皮膚再生
安全性高，適合各年齡層使用，無光敏性。
160
佛手柑
Citrus bergamia
主要化學成分
檸烯 (37%)、乙酸沉香酯 (24~35%)、沉香醇 (4~26%)、γ-松油烯、佛手柑內酯
萃取部位 / 科屬
果皮（冷壓） / 芸香科 Rutaceae
主要功效
抗憂鬱、提振情緒、消炎止痛、抗菌、助消化、平衡皮脂
抗憂鬱提振情緒消炎助消化
含呋喃香豆素（佛手柑內酯），具強烈光敏性。塗抹後 12 小時內避免日曬。可選用無佛手柑內酯版本 (FCF)。
161
小飛蓬
Erigeron canadensis
主要化學成分
D-檸烯 (68%)、母菊酯 (4.5%)、反式-β-羅勒烯 (3.5%)、薄荷三烯異構物
萃取部位 / 科屬
全草（蒸餾） / 菊科 Asteraceae
主要功效
消炎、止血、利尿、驅蚊蟲、促進傷口癒合、收斂
消炎止血驅蟲
菊科過敏者慎用。孕婦避免使用。建議低劑量使用。
162
崢角白梅
Baeckea frutescens
主要化學成分
α-蒎烯、β-蒎烯 (約60%)、檸烯、1,8-桉油醇 (7%)、沉香醇、茴香腦
萃取部位 / 科屬
枝葉（蒸餾） / 桃金娘科 Myrtaceae
主要功效
抗菌、抗炎、抗氧化、驅蚊蟲、皮膚保護、舒緩風濕痠痛
抗菌抗炎驅蟲
一般安全，建議稀釋使用。敏感肌膚先做貼布測試。
163
玫瑰尤加利
Eucalyptus rhodantha
主要化學成分
乙酸香葉酯 (高含量)、香葉醇、1,8-桉油醇、α-蒎烯
萃取部位 / 科屬
葉片（蒸餾） / 桃金娘科 Myrtaceae
主要功效
抗菌抗病毒、舒緩呼吸道、鎮靜安撫、皮膚護理、淨化空氣
呼吸道抗菌花香調
一般安全，適合擴香使用。嬰幼兒及癲癇患者謹慎使用。
164
黃葵
Abelmoschus moschatus
主要化學成分
乙酸法尼酯 (58%)、癸酸乙酯 (4.8%)、十二酸乙酯 (2.4%)、法尼醇
萃取部位 / 科屬
種子（蒸餾） / 錦葵科 Malvaceae
主要功效
抗菌、抗氧化、鎮靜、調節荷爾蒙、天然麝香替代品、皮膚潤澤
麝香調抗菌荷爾蒙
一般安全，適合香水調配及低劑量芳療使用。
165
真正薰衣草
Lavandula angustifolia
主要化學成分
乙酸沉香酯 (25~47%)、沉香醇 (20~40%)、乙酸薰衣草酯、β-丁香烯
萃取部位 / 科屬
花穗（蒸餾） / 唇形科 Lamiaceae
主要功效
鎮靜安眠、皮膚修復、抗焦慮、消炎止痛、平衡情緒、萬用精油
助眠皮膚修復萬用入門首選
芳療界安全性最高的精油之一。2 歲以上兒童可用，孕期中後期低劑量安全。
166
醒目薰衣草
Lavandula x intermedia
主要化學成分
沉香醇 (24~35%)、乙酸沉香酯 (20~32%)、樟腦 (6~10%)、1,8-桉油醇
萃取部位 / 科屬
花穗（蒸餾） / 唇形科 Lamiaceae
主要功效
肌肉放鬆、消炎止痛、舒緩呼吸道、傷口癒合、鎮靜但較提振
肌肉放鬆消炎呼吸道
一般安全。含樟腦成分，癲癇患者及嬰幼兒謹慎使用。鎮靜效果不如真正薰衣草。
167
檸檬薄荷
Mentha citrata
主要化學成分
乙酸沉香酯 (2~45%)、沉香醇 (34~77%)、α-松油醇、β-丁香烯
萃取部位 / 科屬
全草（蒸餾） / 唇形科 Lamiaceae
主要功效
鎮靜安撫、抗痙攣、消炎、抗菌、助消化、舒緩壓力性頭痛
鎮靜抗痙攣清新柑橘調
安全性高，性質溫和。化學結構類似薰衣草而非傳統薄荷，無涼感刺激。
168
含笑
Michelia figo
主要化學成分
苯甲酸甲酯、乙酸苯酯、大根香葉烯 D、β-欖香烯、丁香油烯
萃取部位 / 科屬
花朵（溶劑萃取） / 木蘭科 Magnoliaceae
主要功效
鎮靜安神、抗菌、改善呼吸道、舒緩焦慮、護膚美白、調理情緒
花香調安神護膚
一般安全，建議低劑量使用。花朵原精濃度高，需適當稀釋。
169
紅香桃木
Myrtus communis (ct. myrtenyl acetate)
主要化學成分
乙酸桃金娘烯酯 (12~33%)、1,8-桉油醇 (20~43%)、α-蒎烯 (10~25%)、沉香醇
萃取部位 / 科屬
葉片與嫩枝（蒸餾） / 桃金娘科 Myrtaceae
主要功效
祛痰化痰、抗菌、甲狀腺調節、皮膚收斂、促進靜脈與淋巴循環
祛痰甲狀腺淋巴循環
安全溫和，兒童友善。紅桃金娘含較高 1,8-桉油醇，嬰幼兒慎用高劑量。
170
水果鼠尾草
Salvia dorisiana
主要化學成分
甲基紫蘇酸酯、沉香醇、乙酸沉香酯、1,8-桉油醇、β-丁香烯
萃取部位 / 科屬
全草（蒸餾） / 唇形科 Lamiaceae
主要功效
抗菌、抗真菌、提振精神、消化促進、放鬆情緒、空間淨化
抗菌果香調提振
較少見的精油品種，建議低劑量使用。孕婦避免使用。
171
快樂鼠尾草
Salvia sclarea
主要化學成分
乙酸沉香酯 (45~75%)、沉香醇 (10~20%)、α-松油醇、乙酸香葉酯、香紫蘇醇
萃取部位 / 科屬
花穗與葉片（蒸餾） / 唇形科 Lamiaceae
主要功效
荷爾蒙平衡（類雌激素）、抗痙攣、催情、抗憂鬱、緩解經期不適
荷爾蒙抗痙攣經期抗憂鬱
含香紫蘇醇具類雌激素作用，荷爾蒙敏感族群（乳癌病史等）慎用。勿與酒精併用（可能增強酒醉感）。孕期避免。
172
蘆爪豆
Spartium junceum
主要化學成分
沉香醇 (26%)、十四烷酸 (23%)、樟腦 (14%)、十二烷酸 (13%)、棕櫚酸
萃取部位 / 科屬
花朵（溶劑萃取） / 豆科 Fabaceae
主要功效
鎮靜安撫、抗菌、護膚保濕、舒緩肌肉緊繃、增添甜美花香基調
花香調鎮靜護膚
原精濃度高，須充分稀釋。敏感肌膚先做測試。孕婦慎用。
