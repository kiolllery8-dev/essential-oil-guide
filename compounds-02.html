COMPOUND CLASS II

香豆素與內酯類

香豆素與內酯類化合物是一類含有內酯環結構的芳香分子。呋喃香豆素（如佛手柑內酯）具有明顯的光敏性，倍半萜內酯（如土木香內酯）則具有強效抗菌與消炎作用。這類成分在繖形科與菊科植物中尤為常見。

單萜烯類 香豆素與內酯 氧化物類 倍半萜烯類 醛類 酯類 脂類 苯基酯類 芳香醛酯 倍半萜酮類 單萜醇類 酚類 倍半萜醇類 單萜烯類

中國當歸
Angelica sinensis
主要化學成分
藁本內酯、正丁烯基苯酞、α-蒎烯、呋喃香豆素
萃取部位 / 科屬
根部蒸餾 / 繖形科
主要功效
補血調經、活血化瘀、潤腸通便、鎮痛安神
調經 補血 光敏
⚠️ 含呋喃香豆素，具光敏性；塗抹後 12 小時避免日曬；孕婦/經期過多者避免使用

主要化學成分

萃取部位 / 科屬
根部蒸餾 / 繖形科
主要功效

調經 鎮靜 光敏
⚠️ 含呋喃香豆素，具光敏性；塗抹後避免日曬；孕婦避免使用
56
芹菜籽
Apium graveolens
主要化學成分
苯酞類（丁基苯酞、瑟丹酸內酯）、檸檬烯、β-月桂烯
萃取部位 / 科屬
種子蒸餾 / 繖形科
主要功效
排毒利尿、助消化、降血壓、抗痙攣、淡斑
排毒 消化 需謹慎
⚠️ 可能含微量光敏成分；孕婦避免使用；腎病患者謹慎使用
57
辣根
Armoracia rusticana
主要化學成分
異硫氰酸烯丙酯、苯乙基異硫氰酸酯、微量內酯類
萃取部位 / 科屬
根部蒸餾 / 十字花科
主要功效
強力抗菌、促進血液循環、祛痰、緩解關節疼痛
抗菌 循環 高風險
🚫 含異硫氰酸酯，刺激性極強；可灼傷皮膚與黏膜；僅限極低劑量專業使用；孕婦/兒童/敏感肌膚絕對禁用
58
蒼朮
Atractylodes lancea
主要化學成分
蒼朮素（蒼朮醇）、蒼朮酮、β-桉葉醇、茅朮醇
萃取部位 / 科屬
根莖蒸餾 / 菊科
主要功效
燥濕健脾、祛風散寒、明目、抗菌消炎
健脾 除濕 需謹慎
⚠️ 性質燥烈，陰虛火旺者慎用；孕婦避免使用
59
薑黃
Curcuma longa
主要化學成分
薑黃酮、芳香薑黃酮、α-薑黃烯、β-倍半水芹烯
萃取部位 / 科屬
根莖蒸餾 / 薑科
主要功效
強力消炎、抗氧化、促進肝膽代謝、緩解關節疼痛、美白淡斑
消炎 抗氧化 護膚
⚠️ 可能染色皮膚與衣物；孕婦/膽道阻塞患者避免使用；抗凝血劑使用者謹慎
60
蛇床子
Cnidium monnieri
主要化學成分
蛇床子素（歐前胡素）、異歐前胡素、佛手柑內酯、香豆素類
萃取部位 / 科屬
果實蒸餾 / 繖形科
主要功效
抗真菌、止癢、溫腎壯陽、殺蟲、濕疹調理
皮膚 抗真菌 高風險
🚫 含高量呋喃香豆素，光敏性極強；塗抹後嚴禁日曬；孕婦/兒童禁用；僅供外用
61
新幾內亞厚殼桂
Cryptocarya massoy
主要化學成分
馬索亞內酯（C-10內酯、C-12內酯）、苯甲醛
萃取部位 / 科屬
樹皮蒸餾 / 樟科
主要功效
強力抗菌、抗真菌、促進消化、緩解腹瀉、溫暖身體
抗菌 消化 高風險
🚫 馬索亞內酯具強烈皮膚刺激性，可灼傷皮膚；必須高度稀釋（<1%）使用；孕婦/兒童/敏感肌膚禁用
62
阿魏
Ferula assa-foetida
主要化學成分
二硫化物類、阿魏酸、繖形酮、香豆素、呋喃香豆素
萃取部位 / 科屬
樹脂蒸餾 / 繖形科
主要功效
驅風排氣、助消化、抗痙攣、鎮靜、抗寄生蟲
消化 驅風 氣味強烈
⚠️ 氣味極為強烈（硫化物）；含香豆素類可能有光敏性；孕婦避免使用
63
土木香
Inula helenium
主要化學成分
土木香內酯、異土木香內酯、土木香醇、α-蒎烯
萃取部位 / 科屬
根部蒸餾 / 菊科
主要功效
強力祛痰、抗菌（尤其抗結核桿菌）、促進呼吸道健康、消炎
呼吸道 抗菌 高風險
🚫 倍半萜內酯為強致敏原，極易引發接觸性皮膚炎；需高度稀釋；孕婦/兒童/敏感肌膚禁用
64
大花土木香
Inula graveolens
主要化學成分
龍腦乙酸酯、龍腦、土木香內酯、樟腦
萃取部位 / 科屬
花朵蒸餾 / 菊科
主要功效
強力化痰排黏液、支氣管炎、鼻竇炎、抗卡他性炎症
呼吸道 化痰 需謹慎
⚠️ 倍半萜內酯可能致敏；菊科過敏者避免使用；孕婦/嬰幼兒避免使用
65
川芎
Ligusticum chuanxiong
主要化學成分
藁本內酯、川芎嗪、丁基苯酞、新蛇床內酯
萃取部位 / 科屬
根莖蒸餾 / 繖形科
主要功效
活血行氣、祛風止痛、緩解頭痛、促進血液循環
活血 止痛 需謹慎
⚠️ 活血力強，孕婦/經期過多者/出血傾向者禁用；抗凝血劑使用者謹慎
