## ✦ 快速答案

佛手柑（*Citrus bergamia*）為**芸香科**柑橘類精油，由果皮**冷壓萃取**而得。主要化學成分為**檸烯（37%）、乙酸沉香酯（24-35%）、沉香醇（4-26%）**。香氣明亮甜中帶苦，常見於日常香氛、空間擴香與按摩稀釋（1-3%）。含**呋喃香豆素**（佛手柑內酯），具強烈光敏性，塗抹後 12 小時內應避免日曬。

---

## 一、佛手柑是什麼？

佛手柑精油萃取自佛手柑樹（*Citrus bergamia*）的果皮，這種柑橘以義大利南部 Calabria 為主要產地。與我們食用的水果不同，佛手柑果實本身苦澀，極少直接食用，但其果皮蘊含的精油是香水工業（如著名的 Earl Grey 紅茶香、Eau de Cologne 古龍水基底）與芳療界的明星原料之一。

**植物學分類**

| 項目 | 內容 |
|---|---|
| 中文名 | 佛手柑 |
| 拉丁學名 | *Citrus bergamia* |
| 植物科屬 | 芸香科 Rutaceae |
| 萃取部位 | 果皮（冷壓） |
| 化學分類 | 脂類（單萜烯 + 酯類） |
| 主要產地 | 義大利 Calabria、象牙海岸、巴西 |

## 二、化學成分輪廓

佛手柑屬於「**酯類豐富的柑橘**」，與一般高比例檸烯的甜橙、檸檬不同：

| 化學成分 | 比例 | 香氣與功能輪廓 |
|---|---|---|
| 檸烯（D-Limonene） | ~37% | 柑橘明亮前調 |
| 乙酸沉香酯 Linalyl acetate | 24-35% | 花香、放鬆感（與薰衣草、快樂鼠尾草共有） |
| 沉香醇 Linalool | 4-26% | 柔和花香 |
| γ-松油烯 | 5-8% | 清新木質尾調 |
| 佛手柑內酯（Bergaptene） | 0.2-0.5% | **光敏性來源**（呋喃香豆素類） |

> 「酯類佔比高」是佛手柑與其他柑橘最大的化學差異。這也說明為什麼佛手柑香氣比甜橙更「柔和、複雜」，常被用於高級香水的中調。

## 三、5 種日常香氛用法

### 1. 空間擴香（睡前 / 工作放鬆）
- 100ml 水 + 5-8 滴佛手柑
- 搭配建議：佛手柑 3 + 真正薰衣草 3 + 苦橙葉 2

### 2. 滾珠瓶（隨身使用）
- 10ml 基底油（建議荷荷芭油）+ 2-3 滴佛手柑（即 1-1.5% 稀釋）
- 塗抹手腕、太陽穴；**白天使用後請避免日曬部位**

### 3. 按摩油（肩頸放鬆）
- 30ml 基底油 + 9-12 滴（1.5-2% 稀釋）
- 搭配建議：佛手柑 + 依蘭 + 雪松

### 4. 香水基底
- 佛手柑常作為香水的「前-中調橋接」
- 與薰衣草、玫瑰、橙花、廣藿香、檀香都搭配良好

### 5. 香氛蠟燭 / 室內噴霧
- 100ml 水 + 1ml 乳化劑 + 15-20 滴佛手柑
- 噴於布料、空間，**避免直接噴皮膚**

## 四、光敏性與 12 小時避光原則 ⚠️

這是佛手柑**最重要的安全議題**。

| 議題 | 細節 |
|---|---|
| 為什麼會光敏 | 含呋喃香豆素類成分 Bergaptene（佛手柑內酯），與肌膚接觸後遇紫外線會產生光化學反應 |
| 反應症狀 | 色素沉積（深褐色斑塊）、紅斑、嚴重者起水泡 |
| 安全原則 | 塗抹後 **12 小時內**避免該部位接觸陽光 |
| 安全濃度 | 臉部塗抹建議 ≤ 0.4%；身體建議 ≤ 2% |
| 替代選擇 | 「FCF 佛手柑」（Furanocoumarin-Free / Bergaptene-Free）已去除光敏成分 |

> ✅ **建議夜間使用一般佛手柑**；若需白天塗抹，請選擇 FCF 版本。

---

## 五、FCF（無光敏）版本是什麼？

FCF = **Furanocoumarin-Free**，意指已去除呋喃香豆素類（含 Bergaptene）的佛手柑精油。透過**真空蒸餾**或**冷壓 + 過濾**製程移除光敏成分，保留主要香氣分子。

| 比較項 | 一般佛手柑 | FCF 佛手柑 |
|---|---|---|
| Bergaptene 含量 | 0.2-0.5% | < 0.001% |
| 光敏性 | 高 | 幾乎無 |
| 香氣 | 較完整、複雜 | 略缺尾韻 |
| 適合場合 | 夜間使用、擴香、香水原料 | 日間塗抹、護膚配方 |
| 價格 | 較低 | 較高 |

## 六、與哪些精油搭配最好？

佛手柑的「酯類花香」讓它幾乎和所有精油都能融合，以下是經典搭配：

| 主題 | 搭配 | 比例參考 |
|---|---|---|
| 睡前放鬆 | 佛手柑 + 真正薰衣草 + 羅馬洋甘菊 | 3:3:1 |
| 情緒支持 | 佛手柑 + 苦橙葉 + 依蘭 | 3:2:1 |
| 提神工作 | 佛手柑 + 迷迭香 + 葡萄柚 | 3:2:2 |
| 浪漫氛圍 | 佛手柑 + 玫瑰 + 廣藿香 | 4:1:1 |
| 男士香氛 | 佛手柑 + 雪松 + 杜松 | 3:2:1 |

## 七、常見問題 FAQ

### Q1：佛手柑精油可以擦臉嗎？
建議晚間使用且稀釋至 **0.4% 以下**（10ml 基底油加 1 滴）。日間請選 FCF 版本，或塗抹於不會接觸陽光的部位。

### Q2：佛手柑可以泡澡嗎？
可以。**請先**用 10ml 全脂牛奶、無香沐浴乳或基底油稀釋 3-6 滴後再加入浴缸；切勿直接滴入水中。

### Q3：孕婦可以用佛手柑嗎？
孕期初期（0-12 週）建議避免任何精油使用。中後期可低劑量擴香（0.5-1%），但請先諮詢產科醫師或合格芳療師。

### Q4：佛手柑與佛手柑內酯的差別？
「佛手柑」指整支精油；「佛手柑內酯」(Bergaptene) 指其中具光敏性的單一成分，是 FCF 製程要去除的目標。

### Q5：佛手柑可以擴香整夜嗎？
不建議。擴香 30 分鐘休 30 分鐘是較理想的節奏；長時間密閉空間擴香可能造成嗅覺疲勞與空間刺激。

### Q6：開封後能放多久？
柑橘類精油開封後建議 6-12 個月內用完，因為檸烯易氧化。請冷藏保存延長壽命。
