酯類 Aldehydes (Citral-rich)

本頁收錄富含檸檬醛（Citral）與香茅醛（Citronellal）的精油。這類醛類化合物賦予精油清新的檸檬香氣，具有優異的抗菌、抗真菌、抗發炎與驅蟲功效。檸檬醛由牻牛兒醛（Geranial）與橙花醛（Neral）兩種異構體組成，是天然香料工業的重要原料。

檸檬香桃木
Backhousia citriodora
主要化學成分
牻牛兒醛 (Geranial, 52%)、橙花醛 (Neral, 38%)；合計檸檬醛 (Citral) 含量 90-98%
萃取部位 / 科屬
葉片 / 桃金孃科 (Myrtaceae)
主要功效
強效抗菌、抗真菌、抗病毒、抗氧化、抗生物膜；檸檬醛含量為所有精油之冠（超越檸檬香茅）
抗菌之王抗真菌抗氧化澳洲原生
高檸檬醛含量可能引起皮膚敏感。建議稀釋至 1% 以下使用，敏感肌膚先做測試。避免用於眼周。
140
泰國青檸葉
Citrus hystrix
主要化學成分
L-香茅醛 (L-Citronellal, 58-81%)、香茅醇 (Citronellol, 10%)、3-蒈烯、莎賓烯、沉香醇
萃取部位 / 科屬
葉片 / 芸香科 (Rutaceae)
主要功效
抗菌（對呼吸道病原菌效果顯著）、抗氧化、抗白血病細胞、鎮咳、驅蟲；東南亞傳統藥用與烹飪植物
抗菌呼吸道驅蟲東南亞
香茅醛可能引起皮膚敏感。建議稀釋使用，避免接觸眼睛與黏膜。孕婦慎用。
141
檸檬葉
Citrus limon (Petitgrain)
主要化學成分
檸檬烯 (Limonene)、牻牛兒醛 (Geranial)、橙花醛 (Neral)、月桂烯 (Myrcene)、橙花醇乙酸酯、沉香醇
萃取部位 / 科屬
葉片與嫩枝 / 芸香科 (Rutaceae)
主要功效
抗菌、抗氧化、鎮靜放鬆、提振精神、空氣淨化、抗癌細胞活性（體外研究）
抗菌鎮靜提振精神空氣淨化
一般安全。與檸檬果皮精油不同，葉片精油光敏性較低。建議稀釋至 2% 以下使用。
142
檸檬香茅
Cymbopogon citratus
主要化學成分
檸檬醛 (Citral = Geranial + Neral, 65-85%)、β-月桂烯、牻牛兒醇 (Geraniol)、牻牛兒醇乙酸酯、香茅醛
萃取部位 / 科屬
全株草葉 / 禾本科 (Poaceae)
主要功效
強效抗真菌、抗菌、抗病毒、抗癌、抗氧化、消炎、驅蟲；GRAS 認證安全食品添加物
抗真菌抗菌驅蟲GRAS認證
高檸檬醛含量可能刺激敏感肌膚。建議稀釋至 0.7% 以下使用。孕婦慎用。避免用於受損肌膚。
143
爪哇香茅
Cymbopogon winterianus
主要化學成分
香茅醛 (Citronellal, 30-55%)、牻牛兒醇 (Geraniol, 14-53%)、香茅醇 (Citronellol, 8-16%)、異胡薄荷醇、檸檬烯
萃取部位 / 科屬
全株草葉 / 禾本科 (Poaceae)
主要功效
驅蚊驅蟲（天然植物性防蚊）、抗菌、消炎、除臭、提振精神；香料與日化工業重要原料
天然驅蚊抗菌除臭香料原料
可能引起皮膚敏感。稀釋後使用，避免高濃度直接塗抹。孕婦及嬰幼兒慎用。
144
檸檬尤加利
Corymbia citriodora
主要化學成分
香茅醛 (Citronellal, 56-80%)、香茅醇 (Citronellol, 5-8%)、新異胡薄荷醇 (neo-Isopulegol, 5.5%)、香茅醇乙酸酯
萃取部位 / 科屬
葉片 / 桃金孃科 (Myrtaceae)
主要功效
驅蚊防蟲（WHO 認可有效成分 PMD）、抗真菌、消炎止痛、退燒；與一般尤加利不同，桉油醇含量極低
驅蚊防蟲消炎止痛抗真菌
高香茅醛可能刺激皮膚。建議稀釋使用，避免用於臉部與眼周。3 歲以下兒童慎用。
145
史密斯尤加利
Eucalyptus smithii
主要化學成分
1,8-桉油醇 (1,8-Cineole / Eucalyptol, 70-81%)、α-蒎烯、檸檬烯、水芹烯、松油醇、香茅醛、球紅素醇
萃取部位 / 科屬
葉片 / 桃金孃科 (Myrtaceae)
主要功效
呼吸道暢通、祛痰化痰、抗菌抗病毒、抗真菌、肌肉舒緩；所有尤加利品種中最溫和的一種
呼吸道溫和抗菌兒童適用
所有尤加利中最溫和的品種，適合兒童與老年人使用。仍需稀釋，避免內服。嬰幼兒避免直接塗抹臉部。
146
檸檬細籽
Leptospermum petersonii
主要化學成分
檸檬醛 (Citral = Geranial + Neral, 70-80%)、香茅醛 (Citronellal)、蒎烯
萃取部位 / 科屬
葉片 / 桃金孃科 (Myrtaceae)
主要功效
抗菌、抗真菌、抗病毒、防腐殺菌、驅蚊驅蟲；澳洲原生檸檬茶樹，高檸檬醛含量
抗菌抗真菌驅蟲澳洲原生
高檸檬醛可能引起皮膚敏感。建議稀釋至 1% 以下，敏感肌膚先做局部測試。
147
檸檬馬鞭草
Aloysia citrodora
主要化學成分
牻牛兒醛 (Geranial, 19-29%)、橙花醛 (Neral, 15-20%)、檸檬烯 (5%)、刮花木醇 (Spathulenol, 7-9%)、石竹烯氧化物、菖蒲醇
萃取部位 / 科屬
葉片 / 馬鞭草科 (Verbenaceae)
主要功效
鎮靜安神、抗焦慮、助眠、消炎、抗氧化、促進消化、抗增殖（體外研究）；被譽為「精油中的香檳」
鎮靜安神助眠抗焦慮珍貴精油
可能具光敏性。使用後避免日曬。含檸檬醛可能引起皮膚敏感，建議稀釋使用。價格昂貴，注意摻假品。
148
山雞椒
Litsea cubeba
主要化學成分
檸檬醛 (Citral = Geranial + Neral, 至85%)、牻牛兒醇 (Geraniol)、沉香醇 (Linalool)、檸檬烯
萃取部位 / 科屬
果實 / 樟科 (Lauraceae)
主要功效
抗菌殺菌、抗真菌、消炎、提振精神、抗憂鬱、促進消化、驅蟲；中國傳統藥材「山蒼子」
抗菌提振精神抗憂鬱中藥材
高檸檬醛可能引起皮膚敏感，建議皮膚使用稀釋至 0.8% 以下。檸檬烯的存在可減緩敏感風險。青光眼患者避免口服。
149
蜂蜜香桃木
Melaleuca teretifolia
主要化學成分
檸檬醛 (Citral = Neral + Geranial)、香茅醇 (Citronellol)、牻牛兒醇 (Geraniol)、桉油醇 (Cineole)、β-月桂烯、沉香醇
萃取部位 / 科屬
花朵、葉片與嫩枝 / 桃金孃科 (Myrtaceae)
主要功效
抗菌、抗真菌、消炎、鎮靜放鬆、皮膚保養；Melaleuca 家族中檸檬醛含量最高的品種
抗菌鎮靜皮膚保養澳洲原生
含檸檬醛可能引起皮膚敏感。建議稀釋使用，敏感肌膚先做局部測試。
150
香蜂草
Melissa officinalis
主要化學成分
香茅醛 (Citronellal, ~10%)、牻牛兒醛 (Geranial)、橙花醛 (Neral)；合計醛類約 50-70%。β-石竹烯、大根香葉烯 D
萃取部位 / 科屬
開花全株地上部分 / 唇形科 (Lamiaceae)
主要功效
鎮靜安神、抗焦慮、助眠、抗病毒（尤其針對唇疱疹病毒 HSV）、降血壓、促進消化、抗氧化
鎮靜安神抗病毒助眠珍貴精油
GRAS 認證安全。但高劑量可能影響肝腎功能。含檸檬醛可能引起皮膚敏感，需稀釋使用。價格極昂貴，注意摻假。
151
檸檬桉粉
Eucalyptus staigeriana
主要化學成分
檸檬烯 (Limonene, 7-32%)、檸檬醛 (Citral, 8-12%)、松油烯 (Terpinolene, 7-9%)、1,8-桉油醇 (8-12%)、沉香醇 (4-6%)
萃取部位 / 科屬
葉片 / 桃金孃科 (Myrtaceae)
主要功效
抗感染、抗病毒、防腐殺菌、祛痰、消炎、促進消化；帶有果香檸檬與迷迭香氣息的獨特尤加利
抗感染祛痰消炎澳洲原生
相對溫和。建議稀釋使用，敏感肌膚先做局部測試。避免內服。
152
紫蘇
Perilla frutescens
主要化學成分
紫蘇醛 (Perillaldehyde, 50-60%) 或紫蘇酮 (Perilla ketone, ~81%，依品種而異)、檸檬烯、β-石竹烯、沉香醇
萃取部位 / 科屬
葉片與全株地上部分 / 唇形科 (Lamiaceae)
主要功效
抗氧化、抗真菌、抗菌、抗發炎、抗癌、抗憂鬱、鎮靜助眠、食品防腐（解魚蟹毒）；東亞傳統藥食兩用植物
抗氧化抗菌鎮靜藥食同源
GRAS 認證安全。但紫蘇醛與紫蘇醇可能引起接觸性皮膚炎。高劑量口服曾有過敏報告。適度使用為宜。
153
馬魯科
Aeollanthus suaveolens
主要化學成分
沉香醇 (Linalool, 31-49%)、(E)-β-法尼烯 ((E)-β-Farnesene, 14-35%)、沉香醇乙酸酯 (Linalyl acetate, 8-16%)、2-癸烯-5-內酯
萃取部位 / 科屬
全株地上部分 / 唇形科 (Lamiaceae)
主要功效
抗氧化、細胞毒性（抗腫瘤潛力）、抗菌、鎮靜安神、抗癲癇；亞馬遜傳統民間草藥（catinga de mulata）
抗氧化抗菌鎮靜亞馬遜草藥
研究資料有限，建議在專業指導下使用。孕婦及兒童避免使用。
