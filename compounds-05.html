醛類 Aldehydes

醛類精油含有豐富的苯丙烯類化合物如甲基醚蒔蘿腦（Estragole）與細辛醚（Asarone），具有獨特的辛香與草本氣息。這類精油在傳統醫學中被廣泛應用於消化系統調理、抗痙攣與驅蟲，但因部分成分具有潛在毒性，使用時需格外謹慎控制劑量。

122
菖蒲
Acorus calamus
主要化學成分
β-細辛醚 (69-96%)、α-細辛醚、菖蒲酮、菖蒲萜烯、芹子烯
萃取部位 / 科屬
根莖 / 天南星科 (Araceae)
主要功效
助消化、解痙攣、抗菌、抑制真菌、降溫退燒、平喘、抗血栓、降血壓；傳統用於治療中樞神經系統疾病、失眠與抑鬱
消化調理抗痙攣神經系統低劑量
含細辛醚（致癌物質），口服可導致抽搐，對肝腎有害。孕婦、孩童禁用，成人稀釋濃度需低於 1.5%。
123
龍艾
Artemisia dracunculus
主要化學成分
甲基醚蒔蘿腦 / 草蒿腦 (Estragole, 73-82%)、檸檬烯、β-羅勒烯、β-蒎烯、甲基丁香酚
萃取部位 / 科屬
全株地上部分 / 菊科 (Asteraceae)
主要功效
助消化、鎮痛、抗發炎、退燒、抗真菌、抗菌、免疫調節、保肝、降血糖、甲狀腺調節
消化調理鎮痛抗發炎含草蒿腦
含高濃度草蒿腦（Estragole），具潛在致癌風險。孕婦及兒童避免使用，皮膚使用需高度稀釋。
124
茴香
Foeniculum vulgare
主要化學成分
反式茴香腦 (trans-Anethole, 68-88%)、草蒿腦 (Estragole)、茴香酮 (Fenchone)、檸檬烯、α-蒎烯
萃取部位 / 科屬
種子 / 繖形科 (Apiaceae)
主要功效
促進消化、抗菌、抗真菌、抗氧化、抗發炎、類雌激素作用、利尿、祛痰
消化調理抗菌類雌激素含草蒿腦
含草蒿腦，具類雌激素作用。孕婦、哺乳期、荷爾蒙敏感體質者避免使用。癲癇患者慎用。
125
金合歡茶樹
Melaleuca decora
主要化學成分
甲基丁香酚 (Methyl eugenol, 92.4%)、少量萜烯類化合物 (3.9%)
萃取部位 / 科屬
嫩枝葉 / 桃金孃科 (Myrtaceae)
主要功效
抗菌、抗真菌、鎮痛、局部麻醉、驅蟲；以苯丙烯類為主的獨特 Melaleuca 精油
抗菌鎮痛甲基丁香酚
含極高濃度甲基丁香酚（92%），屬潛在致癌物。僅限專業人士指導下低劑量使用，孕婦兒童禁用。
126
鹹亞茶樹
Melaleuca cajuputi
主要化學成分
1,8-桉油醇 (1,8-Cineole, 42-65%)、α-松油醇、石竹烯、α-蒎烯、檸檬烯
萃取部位 / 科屬
葉片 / 桃金孃科 (Myrtaceae)
主要功效
強效抗菌、抗病毒、抗真菌、呼吸道暢通、緩解肌肉酸痛、免疫支持、驅蟲
呼吸道抗菌肌肉舒緩免疫支持
一般安全，但高濃度可能刺激皮膚與黏膜。孕婦及嬰幼兒慎用，避免直接塗抹於臉部。
127
肉豆蔻
Myristica fragrans
主要化學成分
莎賓烯 (Sabinene, 29%)、α-蒎烯 (10%)、β-蒎烯 (10.6%)、萜品-4-醇 (9.6%)、肉豆蔻醚 (Myristicin, ~12%)、黃樟素 (Safrole, ~2.3%)
萃取部位 / 科屬
種子（核仁）/ 肉豆蔻科 (Myristicaceae)
主要功效
暖身活血、緩解肌肉關節痛、助消化、抗菌、鎮痛、抗氧化；傳統阿育吠陀用藥
暖身活血肌肉舒緩消化調理含黃樟素
含黃樟素（致癌物 1B 類）與肉豆蔻醚。過量使用可致精神錯亂、譫妄。皮膚及呼吸道敏感者慎用，孕婦禁用。
128
粉紅蓮花
Nelumbo nucifera
主要化學成分
沉香醇 (Linalool)、β-石竹烯、α-松油醇、橙花叔醇 (Nerolidol)、苯乙醇 (Phenethyl alcohol)、荷葉鹼 (Nuciferine)
萃取部位 / 科屬
花朵（溶劑萃取原精）/ 蓮科 (Nelumbonaceae)
主要功效
深度鎮靜安神、抗焦慮、抗發炎、抗痙攣、皮膚修復、冥想輔助；荷葉鹼為多巴胺受體促效劑
鎮靜安神抗焦慮冥想皮膚修復
一般認為安全。原精價格昂貴，市售仿品多，購買時注意辨識。敏感肌膚先做局部測試。
129
熱帶羅勒
Ocimum basilicum ct. methyl chavicol
主要化學成分
甲基醚蒔蘿腦 / 草蒿腦 (Methyl chavicol / Estragole, 70-85%)、沉香醇、1,8-桉油醇
萃取部位 / 科屬
開花全株 / 唇形科 (Lamiaceae)
主要功效
強效抗痙攣、消化調理、抗菌、鎮痛、消炎、肌肉放鬆；適用於腸胃脹氣與經期不適
抗痙攣消化調理鎮痛含草蒿腦
含高濃度草蒿腦，具潛在致癌性與抑制凝血作用。皮膚最大濃度 0.1%。6 歲以下兒童、孕婦及哺乳期禁用。
130
露兜花
Pandanus odoratissimus
主要化學成分
苯乙基甲醚 (2-Phenethyl methyl ether, 37-75%)、萜品-4-醇 (11-19%)、α-松油醇、苯乙醇
萃取部位 / 科屬
雄花 / 露兜樹科 (Pandanaceae)
主要功效
鎮靜安神、抗焦慮、抗菌、催情、頭痛緩解；傳統阿育吠陀藥材，廣泛用於香料與香水工業
鎮靜安神催情傳統藥材
一般安全。建議稀釋後使用，敏感肌膚先進行局部測試。
131
歐白芷
Angelica archangelica
主要化學成分
α-蒎烯 (21-30%)、δ-3-蒈烯 (14-17%)、檸檬烯 (13-16%)、α-水芹烯 (8.7%)；含香豆素與呋喃香豆素（佛手柑內酯 Bergapten）
萃取部位 / 科屬
根部 / 繖形科 (Apiaceae)
主要功效
促進消化、補氣強身、祛痰化痰、抗菌抗真菌、抗焦慮、活血化瘀；傳統歐洲草藥
消化調理補氣祛痰光敏性
含佛手柑內酯，具光敏毒性。使用後 12 小時內避免日曬。建議稀釋濃度不超過 0.8%。孕婦及糖尿病患者避用。
132
平葉歐芹
Petroselinum crispum
主要化學成分
芹菜腦 (Apiole, 30-62%)、肉豆蔻醚 (Myristicin, 9-21%)、β-水芹烯、對薄荷三烯、β-月桂烯
萃取部位 / 科屬
種子或葉片 / 繖形科 (Apiaceae)
主要功效
利尿排毒、促進消化、抗氧化、抗菌、通經活血、緩解經期不適；傳統用於腎臟保健
利尿排毒消化調理抗氧化通經
含芹菜腦與肉豆蔻醚，大劑量具肝毒性與神經毒性。孕婦絕對禁用（強效通經作用）。腎臟疾病患者慎用。
133
洋茴香
Pimpinella anisum
主要化學成分
反式茴香腦 (trans-Anethole, 87-97%)、草蒿腦 (Estragole, 0.5-5%)、茴香醛 (Anisaldehyde)、沉香醇
萃取部位 / 科屬
種子 / 繖形科 (Apiaceae)
主要功效
促進消化、抗痙攣、祛痰止咳、抗菌抗真菌、催乳、類雌激素作用、鎮靜助眠
消化調理祛痰類雌激素抗痙攣
具類雌激素作用，荷爾蒙敏感體質（如乳癌、子宮肌瘤）避免使用。孕婦慎用，高劑量可致麻痺。
134
西部黃松
Pinus ponderosa
主要化學成分
β-蒎烯 (21-55%)、甲基醚蒔蘿腦 (Methyl chavicol, 8-42%)、α-蒎烯 (3-10%)、δ-3-蒈烯、α-松油醇
萃取部位 / 科屬
針葉 / 松科 (Pinaceae)
主要功效
呼吸道暢通、祛痰解鬱、肌肉消炎止痛、抗真菌、抗氧化；原住民傳統用於維生素 C 補充
呼吸道肌肉舒緩抗真菌
部分化學型含較高草蒿腦。一般松科精油可能刺激皮膚，建議稀釋至 2% 以下使用。
135
洋茴香蘿文莎葉
Ravensara anisata
主要化學成分
甲基醚蒔蘿腦 (Methyl chavicol / Estragole, 62-95%)、反式茴香腦 (trans-Anethole, ~20%)、沉香醇、檸檬烯
萃取部位 / 科屬
樹皮或葉片 / 樟科 (Lauraceae)
主要功效
消化調理、抗痙攣、鎮痛、抗菌、抗病毒；產於馬達加斯加的特有種精油
消化調理抗痙攣馬達加斯加高草蒿腦
樹皮油含草蒿腦高達 90-95%，具潛在致癌性。強烈建議僅使用葉片油，且需在專業人士指導下使用。
136
防風
Levisticum officinale
主要化學成分
α-松油醇乙酸酯 (α-Terpinyl acetate, 至70%)、β-水芹烯、(Z)-藁本內酯 (Z-Ligustilide, 37-62% 根油)、α-蒎烯、β-蒎烯
萃取部位 / 科屬
根部或葉片 / 繖形科 (Apiaceae)
主要功效
促進消化、利尿排毒、祛風止痛、抗氧化、抗菌；歐洲傳統草藥，用於關節炎與水腫
消化調理利尿抗氧化歐洲草藥
可能具光敏性（含呋喃香豆素）。使用後避免日曬。孕婦及腎臟疾病患者避用。
137
甜萬壽菊
Tagetes lucida
主要化學成分
草蒿腦 (Estragole / Methyl chavicol, 95-97%)、香豆素、槲皮素、類黃酮
萃取部位 / 科屬
開花全株地上部分 / 菊科 (Asteraceae)
主要功效
消化調理、抗痙攣、抗氧化、抗發炎、抗菌；墨西哥傳統藥用植物（Pericón），可替代龍艾使用
消化調理抗痙攣抗氧化極高草蒿腦
含極高濃度草蒿腦（95-97%），為所有精油中草蒿腦含量最高者。具潛在致癌風險，僅限極低劑量外用，孕婦兒童絕對禁用。
